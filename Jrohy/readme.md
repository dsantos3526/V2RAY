#Multi V2ray


Install
```
source <(curl -sL https://multi.netlify.app/v2ray.sh)
```
Keep profile to update
```
source <(curl -sL https://multi.netlify.app/v2ray.sh) -k
```
Uninstall
```
source <(curl -sL https://multi.netlify.app/v2ray.sh) --remove
```
